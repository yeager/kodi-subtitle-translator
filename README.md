# Kodi Subtitle Translator

Automatically translate embedded subtitles in media files to your preferred language.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install kodi-subtitle-translator
```

### Fedora/RPM
```bash
sudo dnf install kodi-subtitle-translator
```
